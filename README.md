# Liberica JDK

BellSoft提供的OpenJDK版本 特点是包含了JavaFX内容
Windows 安装包自动配置环境变量，并且自动关联 jar 打开方式
国内直连下载速度很快(AdoptOpenJDK出来挨打) 所以提供链接

[下载Liberica JDK](https://bell-sw.com/)
